# CasinoWow

## UI Guidelines

- Avoid ALL CAPS for headers, category labels, and general text — use title case or sentence case instead. Only use ALL CAPS when absolutely necessary.
- Avoid border outlines on containers — don't use colored or thick borders to frame cards/sections. If a border is truly needed, use a very thin subtle grey only.
- Avoid line separators. Only use them when absolutely necessary (e.g. separating rows in a table/list). When used, they must not span full width — leave horizontal margins on both sides.
- Avoid excessive emojis. Prefer line icons (e.g. from Tabler Icons) where an icon is needed. If no icon is necessary, use nothing.
- Keep font sizes consistent and avoid excessively small text. Use three sizes: large (headers), medium (titles/subtitles), small (body/content). Don't go below the small size.
